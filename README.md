# avro.tarantool.org

## Installation

1. Install `docker` using an instruction on [docker.com](https://docs.docker.com/engine/installation/)
2. Get `docker-compose` tool from [docker.com](https://docs.docker.com/compose/install/)
3. Build and run services using `docker-compose`:

```
docker-compose build
docker-compose create
docker-compose start
```

4. Started nginx will listen on `127.0.0.1:11112`
